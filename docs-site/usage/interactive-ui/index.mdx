# Interactive & Generative UI

<Image alt={'Interactive UI Overview'} cover src={'https://github.com/user-attachments/assets/interactive-ui-cover'} />

Phở Chat mang đến khả năng **tạo ra giao diện tương tác trực tiếp trong cuộc hội thoại** — không chỉ là văn bản markdown thông thường.

## Bao gồm những gì?

<CardGroup cols={2}>
  <Card href="/usage/interactive-ui/interactive-images" icon="image" title="🖼️ Interactive Images">
    Upload ảnh → AI phân tích → tạo hotspots click được. Dùng cho sơ đồ giải phẫu, floor plan, X-ray, biểu đồ.
  </Card>

  <Card href="/usage/interactive-ui/generative-diagrams" icon="chart-network" title="📊 Generative Diagrams">
    Gõ prompt → AI tạo sơ đồ tương tác. 6 loại: Structural, Timeline, Comparison, Process Flow, Map, Simulation.
  </Card>

  <Card href="/usage/interactive-ui/content-visualizer" icon="scroll" title="📄 Content Visualizer">
    Paste URL paper/bài viết → AI biến thành scrollytelling trực quan với animation từng bước.
  </Card>

  <Card href="/usage/interactive-ui/ai-rendering" icon="wand-magic-sparkles" title="🏠 AI Rendering">
    Upload ảnh phòng trống → AI tạo virtual staging. Render floor plan thành phòng 3D.
  </Card>
</CardGroup>

## Hướng dẫn nhanh

1. **Chọn đúng model**: GPT-4o, Claude Sonnet, hoặc Gemini 2.5 Flash — model lớn mới tạo được artifact format
2. **Gõ prompt rõ ràng**: Nêu rõ loại output mong muốn (interactive image, diagram, visualizer)
3. **Click và tương tác**: Sau khi artifact xuất hiện, click vào các vùng để khám phá

## Use Cases theo ngành

| Ngành           | Tính năng phù hợp                 | Ví dụ                                      |
| --------------- | --------------------------------- | ------------------------------------------ |
| 🩺 Y tế         | Interactive Images                | Sơ đồ giải phẫu, X-ray annotation          |
| 🎓 Giáo dục     | Content Visualizer + Diagrams     | Scrollytelling bài giảng, timeline lịch sử |
| 🏠 Bất động sản | AI Rendering + Interactive Images | Virtual staging, floor plan tương tác      |
| 🔬 Nghiên cứu   | Content Visualizer                | Paper → visual explanation                 |
