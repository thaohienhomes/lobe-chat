# Scientific Agents 🤖

Phở Chat cung cấp **8 AI agent chuyên biệt**, mỗi agent được đào tạo sâu về một lĩnh vực khoa học cụ thể. Tìm trong **Discover** → category **Academic**.

## Danh sách Agents

<CardGroup cols={2}>
  <Card title="💊 Drug Discovery Assistant" href="/agents/drug-discovery">
    **Chuyên môn:** Thiết kế thuốc, ADMET prediction, molecular docking, lead optimization.

    **Tools:** RDKit, ChEMBL, DiffDock, DeepChem, PubChem, DrugBank.
  </Card>

  <Card title="🧬 Bioinformatics Researcher" href="/agents/bioinformatics">
    **Chuyên môn:** Phân tích gene expression, single-cell RNA-seq, variant calling, pathway analysis.

    **Tools:** Scanpy, BioPython, gget, BioServices, KEGG.
  </Card>

  <Card title="🏥 Clinical Research Assistant" href="/agents/clinical-research">
    **Chuyên môn:** Clinical trial design, pharmacogenomics, variant interpretation, adverse events.

    **Tools:** ClinicalTrials.gov, ClinVar, ClinPGx, FDA Databases, cBioPortal.
  </Card>

  <Card title="🤖 Data Science & ML" href="/agents/data-science">
    **Chuyên môn:** Statistical analysis, machine learning, time series, data visualization.

    **Tools:** scikit-learn, PyTorch Lightning, statsmodels, Matplotlib, Plotly.
  </Card>

  <Card title="📝 Scientific Writer" href="/agents/scientific-writer">
    **Chuyên môn:** Literature review, manuscript drafting, peer review, citation management.

    **Tools:** PubMed, OpenAlex, bioRxiv, Citation Manager, Venue Templates.
  </Card>

  <Card title="🔬 Protein Engineer" href="/agents/protein-engineer">
    **Chuyên môn:** Protein structure prediction, design, glycoengineering, binding analysis.

    **Tools:** AlphaFold, ESM, UniProt, PDB, InterPro.
  </Card>

  <Card title="⚗️ Materials Scientist" href="/agents/materials-scientist">
    **Chuyên môn:** Materials discovery, computational chemistry, quantum simulation.

    **Tools:** Pymatgen, Astropy, PennyLane, Qiskit, COBRApy.
  </Card>

  <Card title="🧪 Lab Automation Specialist" href="/agents/lab-automation">
    **Chuyên môn:** Liquid handling automation, protocol management, LIMS integration.

    **Tools:** Opentrons, Benchling, Protocols.io, PyLabRobot, Ginkgo Cloud Lab.
  </Card>
</CardGroup>

## Cách sử dụng Agent

<Steps>
  <Step title="Vào Discover">
    Bấm **Discover** trên navigation → chọn category **Academic**.
  </Step>
  <Step title="Chọn Agent">
    Bấm vào agent phù hợp → **Add to Chat**.
  </Step>
  <Step title="Bắt đầu hỏi">
    Hỏi câu hỏi chuyên ngành. Agent sẽ trả lời với kiến thức chuyên sâu của lĩnh vực đó.
  </Step>
</Steps>

<Tip>
  **Kết hợp:** Bạn có thể sử dụng agent chuyên biệt cùng với **Research Mode** hoặc **Deep Research** để có kết quả tốt nhất. Ví dụ: dùng Clinical Research Agent cho phân tích clinical trials, rồi chuyển sang Research Mode để viết systematic review.
</Tip>
