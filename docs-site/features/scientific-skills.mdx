# Scientific Skills 🧪

Phở Chat tích hợp **170+ kỹ năng khoa học** từ dự án [claude-scientific-skills](https://github.com/K-Dense-AI/claude-scientific-skills), cung cấp truy cập vào **250+ cơ sở dữ liệu** và **60+ Python packages** được tối ưu.

## Cách bật

<Steps>
  <Step title="Vào Settings">
    Bấm icon ⚙️ ở góc dưới bên trái → chọn **Tools**.
  </Step>
  <Step title="Bật Phở Scientific Skills">
    Tìm tool **"Phở Scientific Skills"** → toggle **ON**.
  </Step>
  <Step title="Bắt đầu dùng">
    Quay lại chat → hỏi câu hỏi khoa học. AI sẽ tự động áp dụng kiến thức từ skills phù hợp.
  </Step>
</Steps>
