# Changelog — Tháng 3/2026 🚀

## 05/03/2026 — Research Mode Enhancements

### 📄 Upload PDF nghiên cứu
Upload file PDF trực tiếp vào Research Mode → tự động trích xuất nội dung → tìm kiếm bài liên quan trên PubMed + OpenAlex.

### 🛡️ Kiểm tra đạo văn
Quét so sánh tương đồng với **CrossRef database**. Hiển thị điểm số màu sắc (xanh 0-15%, vàng 15-30%, đỏ >30%) và danh sách nguồn trùng lặp. Hỗ trợ nâng cấp Copyleaks API.

### 🎯 Tạo slide trình bày & PPTX Export
Tự động tạo bộ **14 slide học thuật** từ nghiên cứu. 3 themes: Medical 🏥, Academic 🎓, Minimal ✏️. Export file PPTX cho PowerPoint / Google Slides.
