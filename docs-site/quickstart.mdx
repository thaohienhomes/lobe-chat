# Hướng dẫn nhanh ⚡

<Steps>
  <Step title="Tạo tài khoản">
    Truy cập [pho.chat](https://pho.chat) → đăng ký bằng Email hoặc Google. Miễn phí.
  </Step>

  <Step title="Bật Scientific Skills">
    Vào **Settings** → **Tools** → Bật **Phở Scientific Skills**.

    Khi bật tool này, AI sẽ tự động biết cách:
    - Truy cập 250+ cơ sở dữ liệu khoa học (PubMed, ChEMBL, UniProt...)
    - Sử dụng 60+ Python packages (RDKit, Scanpy, scikit-learn...)
    - Thực hiện 15+ tích hợp (Benchling, DNAnexus, Protocols.io...)
  </Step>

  <Step title="Chọn Agent chuyên biệt (tuỳ chọn)">
    Vào **Discover** → chọn agent phù hợp ngành nghiên cứu:

    | Agent | Phù hợp cho |
    |---|---|
    | 💊 Drug Discovery | Thiết kế thuốc, ADMET, docking |
    | 🧬 Bioinformatics | Phân tích gene, single-cell, omics |
    | 🏥 Clinical Research | Thử nghiệm lâm sàng, pharmacogenomics |
    | 🤖 Data Science & ML | Thống kê, machine learning, dự đoán |
    | 📝 Scientific Writer | Viết bài, trích dẫn, literature review |
    | 🔬 Protein Engineer | Thiết kế protein, AlphaFold, cấu trúc |
    | ⚗️ Materials Scientist | Vật liệu, hoá tính toán |
    | 🧪 Lab Automation | Robot phòng lab, LIMS, protocols |
  </Step>

  <Step title="Bắt đầu nghiên cứu">
    ### Cách 1: Research Mode
    Nhập câu hỏi nghiên cứu → AI tự động tìm paper trên PubMed + OpenAlex → phân tích PICO → viết systematic review.

    ### Cách 2: Deep Research
    Nhập chủ đề → hệ thống multi-agent tự động thực hiện toàn bộ quy trình nghiên cứu → xuất báo cáo hoàn chỉnh.

    ### Cách 3: Chat thông thường
    Hỏi câu hỏi liên quan đến nghiên cứu trong chat → AI gợi ý bạn dùng Research Mode hoặc Deep Research.
  </Step>
</Steps>

<Tip>
  **Mẹo:** Khi bạn hỏi về một chủ đề nghiên cứu trong chat, Phở Chat sẽ tự động hiển thị nút gợi ý **"Dùng Research Mode"** hoặc **"Dùng Deep Research"**. Bấm vào là tự động mở cửa sổ nghiên cứu với câu hỏi đã được điền sẵn!
</Tip>
