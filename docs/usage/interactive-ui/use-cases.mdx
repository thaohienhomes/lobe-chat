# 🎯 Use Cases theo Ngành

## 🩺 Y tế & Lâm sàng

Phở Chat với Interactive UI là công cụ đắc lực cho bác sĩ, sinh viên y khoa, và giáo dục bệnh nhân.

### Giải phẫu & Sinh lý

```
Tạo sơ đồ giải phẫu hệ thần kinh trung ương có thể click. Bao gồm: não bộ (4 thùy), não thất, tủy sống, dây thần kinh sọ (12 đôi). Mỗi phần hiện chức năng và triệu chứng khi tổn thương.
```

```
Tạo timeline phát triển phôi thai người từ tuần 1-40. Click từng giai đoạn để hiện: kích thước phôi, organs đang phát triển, và milestones quan trọng.
```

### Giáo dục bệnh nhân

```
Tạo bài giảng trực quan giải thích bệnh tăng huyết áp cho bệnh nhân 60 tuổi. Dùng ngôn ngữ đơn giản, tránh thuật ngữ y khoa. 4 scene: tim hoạt động, áp lực máu, nguyên nhân, cách kiểm soát.
```

```
Visualize quy trình điều trị ung thư vú giai đoạn 2: phẫu thuật → hóa trị → xạ trị → nội tiết. Timeline với mốc thời gian, tác dụng phụ cần lưu ý từng giai đoạn.
```

### Chẩn đoán hình ảnh

```
[Upload X-ray] Tạo interactive image từ X-ray ngực này. Đánh dấu và giải thích: thùy phổi, bóng tim, góc costophrenic, carina, aortic knob. Dùng để giáo dục sinh viên y khoa.
```

## 🎓 Giáo dục & Đào tạo

### Bài giảng STEM

```
Tạo bài giảng trực quan về phản ứng tổng hợp hạt nhân (nuclear fusion). Target: học sinh lớp 12. 5 scene: nguyên tử hydro, plasma, magnetic confinement, phản ứng, năng lượng tạo ra. Kèm quiz cuối mỗi scene.
```

```
Visualize thuật toán sắp xếp nổi bọt (bubble sort) với array 10 phần tử. Animation từng bước swap, highlight các phần tử đang so sánh. Có nút Play/Pause/Step.
```

### Lịch sử & Địa lý

```
Tạo timeline tương tác về Chiến tranh Thế giới thứ 2 (1939-1945). Phân chia theo mặt trận: Pacific, European, African. Click từng sự kiện để xem: ngày tháng, lực lượng tham gia, kết quả, tổn thất.
```

```
Bản đồ Con đường Tơ lụa từ Trung Quốc đến Châu Âu. Click từng điểm dừng để hiện: tên thành phố, thương phẩm trao đổi, và vai trò trong lịch sử thương mại.
```

### Ngôn ngữ & Văn học

```
Tạo sơ đồ cấu trúc câu tiếng Anh (sentence structure) tương tác. Click từng thành phần (subject, verb, object, modifier) để hiện ví dụ và quy tắc ngữ pháp.
```

## 🏠 Bất động sản

### Trình bày bất động sản

```
[Upload floor plan] Tạo interactive floor plan cho căn hộ 2PN này. Click từng phòng để hiện: diện tích (m²), hướng, và gợi ý nội thất phù hợp. Thêm tổng diện tích và highlight điểm mạnh.
```

```
Tạo bản đồ tương tác khu vực Thủ Đức, TP.HCM. Đánh dấu: dự án XYZ (vị trí), trường học, bệnh viện, siêu thị, metro trong bán kính 2km. Click mỗi POI để hiện khoảng cách và thông tin.
```

### Marketing & Sales

```
Tạo comparison diagram giữa 3 dự án căn hộ: Vinhomes, Masteri, The Manor. So sánh: giá/m², vị trí, tiện ích, pháp lý, tiến độ bàn giao. Highlight điểm mạnh từng dự án.
```

```
[Upload ảnh phòng trống] Virtual staging phòng khách view sông này với phong cách luxury hiện đại. Tối màu, dark wood, gold accents, ánh đèn ấm buổi tối. Trông như ảnh quảng cáo cao cấp.
```

## 🔬 Nghiên cứu Khoa học

### Đọc và tóm tắt paper

```
Biến bài systematic review này thành visual summary: [paste abstract/URL]. 
Format: 
1. Research question (PICO)
2. Methodology flowchart (PRISMA)
3. Key findings chart
4. Limitations
5. Clinical implications

Target audience: bác sĩ lâm sàng, không phải nhà nghiên cứu.
```

```
Tóm tắt 5 bài báo về Large Language Models in Healthcare thành 1 knowledge map. Nhóm theo chủ đề: diagnosis support, drug interaction, clinical notes, patient communication, limitations.
```

### Data Visualization

```
Tạo biểu đồ tương tác từ dataset này: [paste CSV data]. 
- Bar chart so sánh theo nhóm
- Click bar để drill-down chi tiết  
- Filter theo dimension
- Export data button
```

```
Tạo simulation mô hình dược động học (pharmacokinetics) của thuốc X. Điều chỉnh: liều dùng, tần suất, chức năng thận. Hiện đồ thị nồng độ thuốc trong máu theo thời gian.
```
