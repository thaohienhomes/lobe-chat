Self-Host Setup Guide
Step 4 of 4

We've generated a custom Docker setup based on your configuration.

📦 Your Configuration Files

docker-compose.yml
version: '3.8'
services:
  openclaw:
    image: openclaw/openclaw:latest
    ports:
      - "18789:18789"
    env_file: .env
    restart: unless-stopped
    volumes:
      - ./data:/app/data
.env
# Generated by pho.chat OpenClaw Setup
OPENCLAW_LLM_PROVIDER=anthropic
OPENCLAW_API_KEY=sk-ant-***
OPENCLAW_MODEL=claude-3-5-sonnet-20241022
TELEGRAM_BOT_TOKEN=7123456789:AAF-abc123
OPENCLAW_SYSTEM_PROMPT="You are a helpful assistant..."

Deployment Steps

1
Download files above to your server
2
Run: docker compose up -d
3
Verify: curl http://localhost:18789/health
4
